Welcome to my portfolio page. Listed below are several projects that I've worked on. They demonstrate my knowledge of front-end development languages such as HTML5, CSS, and JavaScript.

Welcome Weather

welcomeweather.surge.sh

Welcome Weather is a lightweight and simple weather application built with the intention to be used as a web browser's default home page. It automatically retrieve's the weather based upon a user's geolocation data.

Welcome Weather is a one page, multiple view application built using the AngularJS JavaScript framework. The initial view renders as the current day's weather conditions with the other views ranging from 1 to 4 "days from now".

The site demonstrate's the use of Travis CI and automatically deploys to Surge.sh upon successful builds.

Development Tools Used:

  • HTML5
  • CSS3
  • SASS
  • JavaScript
  • AngularJS
  • TravisCI

Anchorman

anchorman-app.surge.sh

Collaborated with Kyle Holliday

Anchorman is intended to be a new social site for news. The site uses an API that feeds a stream of news agencies gathers them together in a single feed. It is a one-page, multi view site developed using AngularJS.

From the main news feed, you are able to mark articles and save them to a list, viewable from a page in the navigation bar. On another view, users are able to build a list of topics and save the list so that articles in the feed are highlighted.

Development Tools Used:

  • HTML5
  • CSS3
  • SASS
  • JavaScript
  • AngularJS

Olympic Adventures

olympic-game.surge.sh

Olympic Adventures was built using the BackboneJS framework and incorporates UnderscoreJS templates. It is a one page, multi view application. On the first view, users enter their name and select a vehicle.

This application is a grid based game where users choose between two vehicles and move towards the next objective. Every time a user changes their position, they lose fuel and each objective they reach replenishes their fuel by a randomized ammount.

Users must choose from one of two vehicles, where one has a larger starting fuel capacity than the second, but suffers from a tradeoff that it uses more fuel each move.

Development Tools Used:

  • HTML5
  • CSS3
  • SASS
  • JavaScript
  • BackboneJS
  • UnderscoreJS
