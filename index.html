Welcome Weather

Welcome Weather is intended to be a lightweight and simple weather application built using AngularJS. Intended to be used as a web browser's default home page, it automatically retrive's the weather based upon a user's geolocation data.

Welcome Weather is a one page, multiple view application. The initial view renders as the current day's weather conditions with the other views ranging from 1 to 4 "days from now".

Anchorman

Collaborated with Kyle Holliday

Anchorman is intended to be a new social site for news. The site uses an API that feeds a stream of news agencies gathers them together in a single feed. It is a one-page, multi view site developed using AngularJS.

From the main news feed, you are able to mark articles and save them to a list, viewable from a page in the navigation bar. On another view, users are able to build a list of topics and save the list so that articles in the feed are highlighted.
